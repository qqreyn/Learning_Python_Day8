# Learning_Python_Day8

Today's topic is functions!

**Goal:** Master function creation with parameters and return values to write reusable code

- **task1:** Write a `greet(name)` function that prints a greeting

- **task2:** Create `add(a, b)` that returns the sum of two numbers

- **task3:** Build `calculator(num1, num2, operation)` that performs +, -, *, /

- **task4:** Write `is_even(number)` that returns True or False

- **task5:** Create `power(base, exponent=2)` using a default parameter
